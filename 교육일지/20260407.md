# HTML
- HTML 기본구조 : 문서유형~<html lang="ko"> ~머리~몸체~</html?
- <head> : 기본 정보가 들어있는 태그
- 태그는 대소문자를 구분하지 않지만, 기본적으로 소문자로 쓴다.
- 태그는 대소문자를 구분하지 않으며, 소문자를 권장한다.
- 시작 태그로 시작하면 반드시 종료 태그로 종료를 한다.
- 파일 확장자는 반드시 html, htm으로 설정할 것
- 문자의 공백은 한 개만 인식을 하고, 공백을 추가하기 위해서는 특수기호(&nbsp;)를 이용해야 한다.

### HTML 실습 (01_text-and-list.html)
- ! + shift+엔터 : HMTL 기본 구조 
- / + Ctrl : 주석 단축키, 주석 구조 <!-- -->
- 태그명 + Tab : 태그 완성
- shift+del : 한줄지우기
- <hr> : 구분선 태그
- <p> : 문단 태그
- <br> : 줄바꿈 태그. 줄바꿈 하고 싶은 위치에 입력. 뜻: break!
- <pre> : 쓴 모양을 그대로 보여주는 태그. 주로 코드를 표현할 때 사용.
- <b>, <strong> : 둘다 굵은 글씨 태그. 
  - <b> : 별다른 중요성 없이, 단순히 글자만 두껍게 보이게 하고 싶을 때 사용
  -<strong> : 정말 중요하거나 강조가 필요함을 컴퓨터(브라우저, 검색 엔진, 화면 낭독기)에 알릴 때 사용
- <i>, <em> : 둘다 이탤릭체(기울임) 글씨 태그. <i>는 시각적으로만, <em>은 브라우저에게 중요한 내용이라고 의미 부여.
- <mark> : 형광펜
- <del> : 취소선

### 테이블
태그
- table: 표의 시작과 끝을 알리는 전체 태두리
- caption : 표 제목
- thead: 머리글. 표의 최상단 제목줄을 감싸는 태그. 표가 길어서 다음 페이지로 넘어가서 인쇄될 때, 제목줄을 반복해서 보여주는 기능 제공
- tbody: 표의 실제 데이터를 감싸는 태그
- tr : table row. 한 행을 의미
- th : table header. 행, 열의 제목. 표 안에 써주면 자동으로 글씨 굵게 표시
- td : table data. 행의 내용
속성
- rowspan="2": 아래칸을 2줄 병합. (2칸 열병합)
- colspan="2" : 오른쪽 칸을 2칸 병합 (2칸 행병합)

### label
- for="user-id", id="user-id" : 아이디 라벨을 클릭하면 input 박스 빈칸으로 커서 이동됨
  -예시) <label for="user-id">아이디: </label>
<input id="user-id" type="text" placeholder="아이디를 입력하세요">

 속성
- name="userId" : 데이터를 서버로 전송할 때 사용하는 데이터의 이름표
  - 예시)<input id="user-id" type="text" name="userId" placeholder="아이디를 입력하세요">
 - 사용자가 입력창에 "haewon"이라고 적고 제출 버튼을 누르면, 컴퓨터는 서버에 userId=haewon이라는 형태로 데이터를 보냅니다.

### 라디오버튼
