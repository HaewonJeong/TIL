### List, Tuple, Dictionary
- List : 순서 있음, 중복 허용, **[] 표기**, 수정 가능
- Tuple : 순서, 중복 허용, 변경x, **() 표기**, 수정 불가
- Dictionary: **{" ", value}** 표기, key : value, Key 중복 X, Value는 중복 허용
- Set : **{}** 표기, 값만 저장한다. 중복이 불가 하다. 

## 파이썬 기본 자료형

### 1) 파이썬 자료형 개요 (01_datatype)
- 기본 자료형: `int`, `float`, `bool`, `str`, `NoneType`
- 컨테이너 자료형: `list`, `tuple`, `set`, `dict`
- 시퀀스형(순서 O, 인덱싱/슬라이싱 O) vs 이터러블형(반복 가능) 개념 정리
- `iter()`, `next()`를 이용한 이터레이터 개념 학습

### 2) 숫자형 (02_number)
- 사칙연산, 몫(`//`), 나머지(`%`), 거듭제곱(`**`)
- 반올림 `round()`, 소수점 자릿수 지정
- `math` 모듈의 `ceil()`(올림), `floor()`(버림) 함수

### 3) 불리언 (03_bool)
- `True`/`False`와 비교 연산자(`==`, `!=`, `>`, `>=`, `<`, `<=`)
- 논리 연산자 `and`, `or`, `not`
- 복잡한 조건식에는 괄호 `()`를 사용해 우선순위를 명확히 하는 습관

### 4) 문자열 (04_str)
- 한 줄/여러 줄 문자열, 개행문자(`\n`) 사용
- 문자열 연산(`+`, `*`), 형변환(`str()`, `int()`)
- 인덱싱/슬라이싱(`[start:end:step]`), 역순 출력(`[::-1]`)
- 주요 메서드: `in`, `replace()`, `strip()`, `upper()`, `lower()`, `capitalize()`, `swapcase()`, `title()`
- 문자열 포매팅 3가지 방식(`%`, `str.format()`, `f-string`) 중 f-string 실습

### 5) 리스트 (05_list)
- 리스트는 **mutable(가변)** 자료형
- 요소 추가: `append()`, `insert()`
- 인덱싱/슬라이싱, 슬라이싱을 이용한 값 일괄 수정/삭제
- 연결(`+`), 반복(`*`) 연산
- `index()`, `count()`, `sort()`(inplace, 반환값 None) vs `sorted()`(반환값 있음)
- `key=` 옵션으로 정렬 기준 변경(예: `key=len`)
- `for`문, `enumerate()`를 이용한 반복순회

### 6) 튜플 (06_tuple)
- 튜플은 **immutable(불변)** 자료형 → 요소 수정 시 오류 발생
- 튜플 리터럴 생성 방법(`()`, `(10,)`, 괄호 생략)
- 튜플 안에 리스트가 있으면 리스트 내부 요소는 수정 가능(불변은 튜플 자체에 한정)
- 튜플을 이용한 변수 값 교환: `x, y = y, x`
- 리스트와 동일한 반복순회 방식(`for`, `enumerate()`)

### 7) 딕셔너리 (07_dict)
- key-value 쌍으로 구성, key는 중복 불가 & immutable 타입만 허용
- 값 조회, 추가/수정, `pop()`으로 삭제
- value에 리스트, 딕셔너리, 함수 등 다양한 타입 저장 가능
- 딕셔너리에 함수를 저장해 `calculator["add"](10, 20)`처럼 호출하는 실습
- **딕셔너리 생성 방법**: `{}`, `dict()`, 키워드 인자(`dict(one=1, two=2)`), 튜플 리스트(`dict([('one',1),('two',2)])`)
- **딕셔너리 API**: `get(key, default)`로 안전하게 값 조회(키가 없어도 에러 없이 기본값 반환)
- 반복순회: `keys()`, `values()`, `items()` (각각 iterable 반환)
- `zip()`으로 여러 리스트를 묶어서 동시에 순회
- `*리스트`로 unpacking 하여 `zip(*data)`로 컬럼 단위 분리

### 8) 세트 (08_set)
- 세트는 **중복을 허용하지 않는** 자료형, `{}`는 빈 세트가 아니라 빈 딕셔너리
- 리스트/튜플을 `set()`으로 변환하면 중복 제거됨
- 요소 추가/삭제: `add()`, `remove()`(없는 값 삭제 시 에러), `discard()`(없어도 에러 없음), `clear()`
- 세트는 순서가 없어 정렬이 필요하면 `sorted()`를 명시적으로 사용
- `for`문으로 반복순회 가능(iterable)
- **집합 연산**: 합집합 `union()`, 교집합 `intersection()`, 차집합 `difference()`, 대칭차집합 `symmetric_difference()`
