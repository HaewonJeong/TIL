
[집에서 브랜치에 올린 코드 실습실 컴퓨터로 pull 해오고 싶은데 어떻게해?
지금 feature/haewon 브랜치에 잇어]

원격 저장소의 최신 상태 업데이트

- git fetch origin
원격에 어떤 변화가 있는지 목록만 먼저 가져옵니다.

현재 브랜치로 코드 당겨오기
- git pull origin feature/haewon

[수정했어. 내 브랜치에 푸쉬하고 PR 만드는 방법좀]

수정한 코드를 내 브랜치에 반영하고, 팀원들에게 검토를 요청하는 PR(Pull Request) 생성까지의 과정을 정리해 드릴게요.

1. 내 브랜치에 코드 푸쉬(Push)하기
터미널이나 인텔리제이 하단 Terminal에서 아래 명령어를 차례대로 입력하세요.

수정된 파일 확인

Bash
git status
수정된 파일을 스테이징(준비 영역)에 올리기

Bash
git add .
변경 사항 기록(커밋)

Bash
git commit -m "feat: 무기 선택 예외 처리 및 로직 보완"
원격 저장소(GitHub)의 내 브랜치로 보내기

Bash
git push origin feature/haewon
