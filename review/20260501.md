### 정렬
- 선택 정렬 : 가장 작은 값은 찾아 맨 앞으로 보내는 과정을 반복하는 배열
```
for(int i = 0 ; i < arr.length -1 ; i++){
            int minIndex = i; //일단 현재 자리(i)에 있는 값이 가장 작다고 가정
            for (int j = i; j < arr.length; j++){
                if(arr[minIndex] > arr[j]) {
                    minIndex = j;
                }
            }
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;

            System.out.println((i+1)+"회차 정렬 후 : "+Arrays.toString(arr));
        }
```
- 버블 정렬 : 인접한 두 원소를 비교하여, 큰 값을 오른쪽으로 계속해서 이동 시키는 배열
```
int[] arr = {2,5,4,6,1,3};

        // i : 정렬이 확정된 영역의 시작 인덱스 (뒤에서 부터)
        for(int i = arr.length - 1; i >= 0; i--){
            //j : 비교할 대상의 인덱스 (앞에서 부터)
            for(int j = 0; j < i; j++){
                if(arr[j] > arr[j+1]){ //앞과 뒤 비교
                    int temp = arr[j];
                    arr[j] = arr[j+1];
                    arr[j + 1] = temp;
                }
                System.out.println((arr.length - i) + "회차 정렬 후 : " + Arrays.toString(arr));
            }
            System.out.println("정렬 후 : "+ Arrays.toString(arr));
        }
```

🚨 에러 원인 1: 문법 오류 (타입 누락)
자바에서 변수를 새로 만들 때는 무조건 타입을 적어줘야 합니다.
작성하신 for (num : numbers)를 보면 num 앞에 이 숫자가 정수인지 뭔지 알려주는 타입이 빠져있습니다.

수정: for (int num : numbers) 로 고쳐야 일단 빨간 줄(컴파일 에러)이 사라집니다.

🚨 에러 원인 2: 진짜 큰 함정 (값 복사)
위의 문법 오류를 고쳐서 실행이 되더라도, 배열 numbers를 출력해 보면 여전히 전부 0만 나올 겁니다. 난수가 하나도 들어가지 않아요!

왜 그럴까요? for (int num : numbers) 문법은 배열의 진짜 공간에 접근하는 게 아니라, 값을 '복사'해서 가져오는 방식이기 때문입니다.

비유하자면: 배열이라는 진짜 '방'에다가 난수를 넣어야 하는데, 방 안에 있던 0이라는 숫자만 num이라는 '바구니'로 복사해 온 겁니다. 그리고 그 바구니(num)에다가 난수를 담아놓고는, 다음 턴이 되면 바구니를 그냥 버려버리는 꼴이 됩니다. 진짜 방은 건드리지도 못한 거죠.

🛠️ 올바른 해결 방법
배열에 새로운 값을 집어넣거나 수정할 때는 반드시 '인덱스(방 번호)'를 직접 찾아가서 넣어야 합니다. 즉, 우리가 아는 기본 for문을 쓰셔야 해요!

Java
Random rd = new Random();

// 인덱스 i를 써서 0번 방부터 끝 방까지 직접 찾아갑니다!
for (int i = 0; i < numbers.length; i++) {
    // 로또니까 6이 아니라 45로 살짝 바꿨습니다! (1~45)
    numbers[i] = rd.nextInt(45) + 1; 
}
```
### 1-45 로또 숫자 생성. 랜덤 함수 두가지.
- int newRdNum = rd.nextInt(45)+1; //rd.nextInt(n)는 0~n까지 랜덤 숫자 생성 
- (int) (Math.Random()*45)+1; // Math.Random()은 0<x<1까지 랜덤 난수 생성
```
