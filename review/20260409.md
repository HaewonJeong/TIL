### 객체
- 자바스크립트는 객체 기반 프로그래밍 언어이다. 
- 원시 값을 제외한 나머지(함수, 배열, 정규 표현식 등)은 모두 객체이다.
- 객체는 0개 이상의 프로퍼티로 구성 된 집합이며 프로퍼티는 키(key)와 값(value)으로 구성된다.
- 자바스크립트에서 사용할 수 있는 모든 값은 프로퍼티 값이 될 수 있다.
- 프로퍼티 값이 함수일 경우 method 라고 부른다.

```
//객체 리터럴
 const student = {
	//키-값 쌍으로 구성 된 프로퍼티. 중괄호를 이용해 객체 리터럴 생성
    // 	//프로퍼티 : 객체의 상태를 나타내는 값(data)
	name: "권판다",
    age: 1,
    // 메서드 : 프로퍼티(상태 데이터)를 참조하고 조작할 수 있는 동작
    //함수가 프로퍼티로 올 경우 메서드라고 부르고, 메서드는 동작을 정의한다.
    //this 키워드는 현재 객체를 가리킨다. 예시에서는 student
    getInfo: function(){
        return `${this.name}(은)는 ${this.age}세 입니다.`
    }
}

console.log(typeof student); //object
console.log(student); //객체안에 정의한 프로퍼티 출력. { name: '권판다', age: 1, getInfo: [Function: getInfo] }

//프로퍼티 없이도 빈 객체 생성 가능
const student2 = {}
console.log(typeof student2);
console.log(student2);


```

### 콜백 함수, 고차 함수, 일급 객체
```
/**
 * 콜백 함수
 * 다른 함수의 '재료(인자)'로 전달되어, 그 함수의 실행 시점을 제어하는 함수이다.
 * "자바스크립트는 함수를 일급 객체로 취급하기 때문에, 
 * add를 콜백 함수로 만들어서 calculator라는 고차 함수에 던져줄 수 있다!"
 */

// 1. 고차 함수 (Higher-Order Function): 함수를 인자로 받는 함수
// calculator 함수
function calculator(calculatorCallback, a, b) { 
    console.log('계산을 시작합니다..');
    const result = calculatorCallback(a, b); //const result = add(10, 5);가 실행되는 것과 같다.
    return result;
}

// 2. 콜백 함수 (Callback Function)
// : add 함수와 multiply 함수 (그리고 calculator 내부에서 이를 받아주는 calculatorCallback 파라미터)
// (참고)일급 객체 (First-class Object) : 자바스크립트에서는 함수가 특별 대우를 받는 게 아니라, 일반 변수나 값과 동등한 자격(일급 시민)을 갖기 때문에
// 이렇게 함수를 다른 함수로 던져주는 게 가능
// 특정 함수의 '이름'이라기보다는, 자바스크립트가 함수를 대하는'취급 방식(성질)
function add(a, b) { //<-- 콜백 함수
    return a + b;
}

function multiply(a, b) {
    return a * b;
}

// 3. 실행 (일급 객체의 성질을 이용)
const addResult = calculator(add, 10, 5);
console.log(addResult);

const multiplyResult = calculator(multiply, 10, 5);
console.log(multiplyResult);


const numbers = [3, 10, 1, 4, 2];

//1. 문제 (문자열 정렬): 기본값이 '사전순'이라, 숫자를 넣어도 앞글자만 보고 10이 2보다 먼저 나오는 오류가 발생함.
const a = numbers.sort();
console.log(a); //[ 1, 10, 2, 3, 4 ] 출력

//2.해결 (콜백 함수 주입): 반드시 (a, b) => a - b를 주입해서, "글자가 아닌 숫자 크기"로 비교하라고 강제해야 함.
numbers.sort(
    function(a, b) {
    // a - b가 음수면 a가 앞으로, 양수면 b가 앞으로 정렬된다.
    // return a - b;   // 오름차순 정렬
    return b - a;   // 내림차순 정렬
}
)
```
